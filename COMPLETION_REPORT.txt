INSTALLATION STEPS:
================================================================

1. Install XAMPP (2 minutes)
   → Download from apachefriends.org
   → Install with defaults
   → Start Apache and MySQL

2. Import Database (1 minute)
   → Open http://localhost/phpmyadmin
   → Import database/schema.sql
   → Database created automatically

3. Copy Files (1 minute)
   → Copy project to: C:/xampp/htdocs/dataoptimize_pro/

4. Configure (30 seconds)
   → Verify .env file settings

5. Access Site (30 seconds)
   → Open: http://localhost/dataoptimize_pro
   → Login with admin credentials

TOTAL TIME: ~5 MINUTES
